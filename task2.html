The task is to perform classification for a5 using an LLM-based transformer. A5's checklist must be generated as an output, and chest radiographs, reports, and Q&A 1 to 4 are used as inputs. The level is divided according to the resource used for input, and it is expected to finally perform classification for a5 through input 1. As mentioned above, this problem has a resource saving effect in the process of creating a dataset as a part of automatic annotation.




We provide code support for the project.

Image encoders used in the project are CheXNet, Inception-V4, and ResNet-152, and text encoders are GPT-2, GPT-3, GPT-3.5, and BERT.


Image Encoder introduces huggingface.

CheXNet: https://github.com/arnoweng/CheXNet
Inception-V4: https://github.com/titu1994/Inception-v4
ResNet: https://github.com/huggingface/pytorch-image-models


Text Encoder introduces huggingface.

BERT: https://huggingface.co/docs/transformers/model_doc/bert
GPT 2: https://huggingface.co/docs/transformers/model_doc/gpt2
GPT 3: https://huggingface.co/transformers/v3.0.2/model_doc/gpt.html
GPT 3.5: https://huggingface.co/transformers/v3.5.1/model_doc/gpt.html
